Install Docker

Download:
https://www.docker.com/products/docker-desktop/
Verify:
copy docker version
Example output:
Client: Docker Engine Server: Docker Engine
---

Create Docker Volumes

copy docker volume create nextcloud_data docker volume create nextcloud_db
---

Create Network

copy docker network create nextcloud_net
---

Start Database

copy docker run -d ^ --name nextcloud-db ^ --network nextcloud_net ^ -e MYSQL_ROOT_PASSWORD=RootPass123 ^ -e MYSQL_DATABASE=nextcloud ^ -e MYSQL_USER=nextcloud ^ -e MYSQL_PASSWORD=NextcloudPass ^ -v nextcloud_db:/var/lib/mysql ^ mariadb:10.6
---

Start Nextcloud

copy docker run -d ^ --name nextcloud ^ --network nextcloud_net ^ -p 8080:80 ^ -v nextcloud_data:/var/www/html ^ -v D:/CloudStorage:/mnt/cloud ^ nextcloud:28
---

Install Cloudflare Tunnel

copy docker run -d ^ --name cloudflared ^ --network nextcloud_net ^ cloudflare/cloudflared ^ tunnel --url http://nextcloud:80
Example output:
https://random-server.trycloudflare.com
