Step 1 — Computer Requirements

  • Windows 10 or Windows 11
  • 64-bit processor
  • 8 GB RAM recommended
  • At least 20–30 GB free on system drive
  • Internet connection
Why system space is needed Docker and Nextcloud require some space to run. This is different from the storage used for your cloud files.

Step 2 — Choose a Drive for Cloud Storage

Example computer layout:
C:\ → Windows D:\ → Cloud Storage
In this guide we will use:
D:\CloudStorage
Your drive letters may look like:
C:\ D:\ E:\
Choose a drive with enough free space.

Step 3 — Verify Drives Using PowerShell

WHERE TO RUN
PowerShell
INPUT
copy Get-PSDrive -PSProvider FileSystem
EXPECTED OUTPUT
Name Used (GB) Free (GB) Root C 90 120 C:\ D 20 380 D:\

Step 4 — Create Cloud Storage Folder

WHERE TO RUN
PowerShell
INPUT
copy mkdir D:\CloudStorage
EXPECTED OUTPUT
Directory: D:\ Mode LastWriteTime Name d---- 06-03-2026 CloudStorage
Verify folder
copy dir D:\CloudStorage

Step 5 — Check Your IP Address

WHERE TO RUN
Command Prompt
INPUT
copy ipconfig
EXPECTED OUTPUT
IPv4 Address : 192.168.29.143 Subnet Mask : 255.255.255.0 Default Gateway : 192.168.29.1

Step 6 — Setup Static IP (WiFi)

Open:
  • Settings
  • Network & Internet
  • Wi-Fi
  • Your connected network
Change:
IP Assignment → Manual
Enable IPv4 and enter example values:
FieldExample
IP Address192.168.29.143
Subnet255.255.255.0
Gateway192.168.29.1
DNS 11.1.1.1
DNS 28.8.8.8
Restart PC and run
ipconfig
If the IP stays the same, static IP is working.

Step 7 — Open PowerShell (Administrator)

  1. Click Start Menu
  2. Search powershell
  3. Right click Windows PowerShell
  4. Select Run as Administrator
Expected screen:
PS C:\WINDOWS\system32>

Step 8 — Install WSL

INPUT
copy wsl --install
Expected output:
Installing: Windows Subsystem for Linux Restart required

Step 9 — Install Node.js

Download from
https://nodejs.org
Verify:
copy node -v
v20.x.x
Check npm:
npm -v

Step 10 — Install Docker Desktop

Download from
https://docker.com/products/docker-desktop
Verify installation:
copy docker --version
Docker version 24.x.x

Step 11 — Create Docker Network

copy docker network create nextcloud_net
Expected:
f8a1b3e6d3b1f7c9e5c8...

Step 12 — Run Nextcloud Container

copy docker run -d ^ --name nextcloud ^ --network nextcloud_net ^ -p 8080:80 ^ -v D:\CloudStorage:/mnt/cloud ^ nextcloud
Verify:
docker ps

Step 13 — Access Your Local Cloud

Open browser:
http://localhost:8080
or
http://192.168.29.143:8080
You should see the **Nextcloud setup screen**.

Stage 1 Complete

Your personal cloud server is now running locally. Next we will connect it to the internet using Cloudflare Tunnel.
