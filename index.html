Step 1 — Computer Requirements

  • Windows 10 or Windows 11
  • 64-bit processor
  • 8 GB RAM recommended
  • 50 GB free storage
  • Stable internet connection
Why these requirements matter Docker containers require virtualization and enough memory to run properly.

Step 2 — Check Your IP Address

WHERE TO RUN
Command Prompt
HOW TO OPEN
  1. Click the Start Menu
  2. Type cmd
  3. Click Command Prompt
INPUT
copy ipconfig
EXPECTED OUTPUT
Ethernet adapter Ethernet IPv4 Address : 192.168.29.143 Subnet Mask : 255.255.255.0 Default Gateway : 192.168.29.1
Restart your PC and run the command again to confirm the IP stays the same.
Why static IP is important Servers should keep the same IP address so services can always reach them.

Step 3 — Open PowerShell (Administrator)

  1. Click the Start Menu
  2. Search for powershell
  3. Right click Windows PowerShell
  4. Select Run as Administrator
EXPECTED SCREEN
Windows PowerShell PS C:\WINDOWS\system32>
What is PowerShell? PowerShell is a command-line interface used to control Windows and run system commands.

Step 4 — Install WSL

WHERE TO RUN
PowerShell (Administrator)
INPUT
copy wsl --install
EXPECTED OUTPUT
Installing: Windows Subsystem for Linux Installing: Virtual Machine Platform Restart required
Restart your computer once installation finishes.

Step 5 — Install Node.js and npm

Download from
https://nodejs.org
Choose
LTS version (x64)
VERIFY INSTALLATION
copy node -v
v20.x.x
copy npm -v
10.x.x

Step 6 — Install Docker Desktop

Download from
https://www.docker.com/products/docker-desktop
After installation restart your computer.
VERIFY DOCKER
copy docker --version
Docker version 24.x.x

Step 7 — Create Docker Network

WHERE TO RUN
PowerShell (Administrator)
INPUT
copy docker network create nextcloud_net
EXPECTED OUTPUT
f8a1b3e6d3b1f7c9e5c8...

Step 8 — Run Nextcloud Container

WHERE TO RUN
PowerShell
INPUT
copy docker run -d ^ --name nextcloud ^ --network nextcloud_net ^ -p 8080:80 ^ nextcloud
VERIFY
copy docker ps
CONTAINER ID IMAGE NAME abc123 nextcloud nextcloud

Step 9 — Access Your Local Cloud

Open your browser and go to
http://localhost:8080
or
http://192.168.29.143:8080
You should see the **Nextcloud setup page**.

Stage 1 Complete

Your personal cloud server is now running locally. Next we will expose the server to the internet using Cloudflare Tunnel.
